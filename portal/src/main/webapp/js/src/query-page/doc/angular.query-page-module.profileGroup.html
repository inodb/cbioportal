Class: profileGroup

angular.query-page-module. profileGroup

new profileGroup()

An angular attribute directive which replaces that element with the given html fragment, which is the markup for a single genomic profile group in the section in which genomic profiles are selected. A profile group is either a single checkbox, or a collection of radio buttons, depending on how many genomic profiles of a given alteration type are available for the specified cancer study. Is very specific, should really only be used in one context.
