(static) createFilterTree(cmds, gene_attrs) → {Object}

Helper function for filterSample. Creates a "filter tree", which is basically the result of visiting a query's parse tree and converting each command into a boolean of whether the given genotype fulfills that requirement.
Parameters:
Name Type Description
cmds Object The attribute commands portion of the result of parseQuery. This is a parse tree.
gene_attrs Object The genotype of a particular gene in a given sample.
Source:
  • lib/oql.js, line 151
See:
  • filterSample
Returns:
A nested object that represents a tree in the following way: Where T is a subtree and B is a boolean literal value, a leaf looks like {type:'LEAF', value:B} and a node looks like one of the following three: • {type: 'NOT', child: T} • {type: 'AND', left: T1, right: T2} • {type: 'OR', left:T1, right:T2}
